# US Industry Financial Performance Analysis
ACC102 Track 2 | Python Data Product

## 1. Project Purpose
This project analyzes and compares the financial performance of US listed companies across four key industries: Technology, Consumer, Healthcare, and Financials. The goal is to identify profitability and risk differences using standard financial ratios.

## 2. Data Source
- Database: WRDS Compustat (US Annual Fundamentals)
- Period: 2023 – 2024
- Access Date: 2026-04-13

## 3. Key Financial Metrics
- ROA (Return on Assets)
- Debt Ratio (Total Debt / Total Assets)
- Gross Profit Margin

## 4. Tools & Libraries
- Python
- Pandas, Matplotlib, Seaborn
- WRDS API

## 5. Main Insights
- Technology industry has the highest ROA and gross margin.
- Financial companies have significantly higher debt ratios.
- Consumer industry shows stable and balanced financial performance.
